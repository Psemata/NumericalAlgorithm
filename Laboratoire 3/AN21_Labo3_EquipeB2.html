Contexte

Le but de ce laboratoire est de développer une application pour mettre en pratique la méthode de Gauss pour résoudre un système linéaire. La lecture des matrices doit pouvoir être faîte à partir d'un fichier JSON au format DimN que l'on va copier dans un text area. Le script doit être testé sur trois matrice.

  • 3x3
  • 3x3 avec rotation de ligne
  • 210x210
Le temps de process est important, un classement des techniques des plus rapides aux plus lentes sera fait pour ajouter ou non des points.

Méthode

Afin de résoudre un système d'équation linéaire avec la méthode par élimination de Gauss, il faut supprimer certains éléments de la matrice pour au final arriver à une matrice permettant d'utiliser la méthode de subtitution. Notre groupe a eu l'idée "d'éviter" la subtitution en transformant la matrice d'équation en matrice triangulaire à 1 pour finalement avoir une solution par ligne. Ainsi, le travail du script créé est de travailler la matrice pour récupérer une forme :
1 0 0 y1
0 1 0 y2
0 0 1 y3
Finalement, c'est à l'aide d'un timer que l'on arrive à voir le temps que le processus à besoin pour calculer ses réponses.

Conclusion

Bien que compliqué à utiliser, la méthode d'élimination de Gauss permet de trouver les différentes solutions d'un système d'équation linéaire sous forme matricielle. L'application de l'équipe 2 a généré ces temps pour les matrices :
3x3 avec swap obligatoire : ~1ms
3x3 avec que des 0 : erreur, impossible à calculer
400x400 : ~165ms
